HTML要素

mark要素

mark要素は、文書内の該当テキストを目立たせる要素である。ハイライト部分はmark要素で指定する。具体的な書き方は以下の通りである。

<mark>ハイライト表示</mark>

ブラウザ上では下のように表示される。

ハイライト表示

progress要素

progress要素は、タスク完了までの進行状況を示す要素である。具体的な書き方は以下の通りである。

<progress></progress>

ブラウザ上では下のように表示される。

またvalue(現在値)とmax(最大値)を使って、現在の進捗状況を示すことも出来る。

<progress value="50" max="100"></progress>
<progress value="100" max="100"></progress>

ブラウザ上では下のように表示される。

del要素

del要素は、後から削除された部分であることを示す要素である。具体的な書き方は以下の通りである。

<del>今週末は晴れるでしょう。</del>

ブラウザ上では下のように表示される。

今週末は晴れるでしょう。低気圧の影響で天気は崩れるようです。(12月20日追記)

q要素

q要素は、その内容が引用であることを示す要素である。短い引用の際に使い、長文になる場合はblockquote要素を用いる。具体的な書き方は以下の通りである。

<q>短い引用</q>

ブラウザ上では下のように表示される。

短い引用

sup要素

sup要素は、上付け文字を表現するための要素である。具体的な書き方は以下の通りである。

2<sup>10</sup> = 1024

ブラウザ上では下のように表示される。

210 = 1024

CSSプロパティ

opacityプロパティ

opacityプロパティは、要素の透明度を指定するためのプロパティである。

透明度1

透明度0.5

これは、「<p id="element12">透明度0.5</p>」という要素に対して、「#element12 { opacity:0.5; background-color: #2ca9e1; }」というスタイルを指定した結果である。

font-sizeプロパティ

font-sizeプロパティは、フォントのサイズを指定するためのプロパティである。

100px

これは、「<p id="element21">100px</p>」という要素に対して、「#element21 { font-size: 100px; }」というスタイルを指定した結果である。

letter-spacingプロパティ

letter-spacingプロパティは、文字と文字の感覚を広げるプロパティである。

文字間を広げることができる

これは、「<p id="element31">文字間を広げることができる</p>」という要素に対して、「#element31 { letter-spacing: 10px; }」というスタイルを指定した結果である。

linear-gradientプロパティ

linear-gradientプロパティは、グラデーションを表示させるプロパティである。

これは、「<div id="element41"></div>」という要素に対して、以下のようにスタイルを指定した結果である。

#element41 {
	width: 500px;
	height: 200px;
	background: linear-gradient(blue,red);
}

text-shadowプロパティ

text-shadowプロパティは、ボックスに影をつけるためのプロパティである。

これは、「<div id="element51">箱に影がつく</div>」という要素に対して、「以下のようにスタイルを指定した結果である。

#element51 {
	width: 500px;
	height: 50px;
	background-color: red;
	box-shadow: 10px 10px 10px;
}
